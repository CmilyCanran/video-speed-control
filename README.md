# 视频倍速播放控制

> 🔄 **倍速列表循环** | ⌨️ **快捷键切换** | 🎯 **手动调节**

一个油猴脚本，支持视频倍速循环播放控制。

## 核心功能

### 🔄 倍速列表循环
- 默认倍速列表：`1 → 1.25 → 1.5 → 1.75 → 2 → 2.5 → 3 → 1...`
- 支持自定义倍速列表，满足不同场景需求

### ⌨️ 快捷键 & 🎯 手动控制
| 操作方式 | 功能 |
|---------|------|
| 按 `` ` `` 键 | 循环切换下一个倍速 |
| 点击 `+` / `-` 按钮 | 手动增减倍速 |
| 点击 🔒 锁定图标 | 锁定当前倍速，防止被网页重置 |
| 按 `Esc` 键 | 关闭设置面板 |

### 🔒 倍速锁定
- 锁定后自动保持设置的速度，防止被网页重置
- 适用于视频网站强制重置倍速的情况

### ⚙️ 智能辅助
- **自动检查**: 自动检测并恢复被网页重置的倍速
- **自动隐退**: 空闲时自动隐藏控制按钮
- **暂停联动**: 暂停视频时自动显示按钮
- **Shadow DOM**: 支持百度网盘等使用 Shadow DOM 的网站

## 安装方法


1. 查看 video-speed-control.js 脚本源码
2. 在扩展中创建新脚本，将代码复制进去
3. 刷新任意视频网站即可使用

## 配置项说明

| 选项 | 默认值 | 说明 |
|------|--------|------|
| 倍速列表 | 1, 1.25, 1.5, 1.75, 2, 2.5, 3 | 自定义倍速 |
| 自动检查 | 开启 | 自动恢复被重置的倍速 |
| 检查间隔 | 5000ms | 自动检查的间隔时间 |
| 自动隐退 | 开启 | 5秒后自动隐藏 |
| 隐退时间 | 5000ms | 隐藏前的等待时间 |
| 载入时显示 | 开启 | 页面加载时显示按钮 |
| 暂停时显示 | 关闭 | 暂停时显示按钮 |
| 暂停时隐退 | 关闭 | 暂停后自动隐藏 |


## 技术原理

1. **playbackRate 劫持**: 通过拦截 HTMLMediaElement.prototype.playbackRate 实现倍速锁定
2. **Shadow DOM 扫描**: 突破 closed 模式的 Shadow DOM 限制
3. **MutationObserver**: 监听 DOM 变化，自动处理动态加载的视频
4. **配置持久化**: 使用 GM_getValue/GM_setValue 保存用户设置
